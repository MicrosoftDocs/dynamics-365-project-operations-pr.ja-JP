# <a name="business-transactions"></a><span data-ttu-id="dc635-103">ビジネス トランザクション</span><span class="sxs-lookup"><span data-stu-id="dc635-103">Business transactions</span></span>

<span data-ttu-id="dc635-104">Dynamics 365 Project Service Automation で *ビジネス トランザクション* は、どのエンティティでも表現されない抽象的な概念です。</span><span class="sxs-lookup"><span data-stu-id="dc635-104">In Dynamics 365 Project Service Automation, *business transaction* is an abstract concept that isn't represented by any entity.</span></span> <span data-ttu-id="dc635-105">ただし、エンティティの一部の一般的なフィールドとプロセスは、ビジネス トランザクションの概念を使用するように設計されています。</span><span class="sxs-lookup"><span data-stu-id="dc635-105">However, some common fields and processes on entities are designed to use the concept of business transactions.</span></span> <span data-ttu-id="dc635-106">次のエンティティはこの抽象化を使用します:</span><span class="sxs-lookup"><span data-stu-id="dc635-106">The following entities use this abstraction:</span></span>

- <span data-ttu-id="dc635-107">見積依頼明細行の詳細</span><span class="sxs-lookup"><span data-stu-id="dc635-107">Quote line details</span></span>
- <span data-ttu-id="dc635-108">契約品目の詳細</span><span class="sxs-lookup"><span data-stu-id="dc635-108">Contract line details</span></span>
- <span data-ttu-id="dc635-109">見積もり行</span><span class="sxs-lookup"><span data-stu-id="dc635-109">Estimate lines</span></span>

<span data-ttu-id="dc635-112">これらのエンティティのうち、見積品目の詳細、契約品目の詳細、見積品目は、プロジェクト ライフサイクルの見積フェーズにマップされます。</span><span class="sxs-lookup"><span data-stu-id="dc635-112">Of these entities, Quote line details, Contract line details, and Estimate lines are mapped to the estimation phase in the project lifecycle.</span></span> <span data-ttu-id="dc635-113">仕訳帳明細行と実績エンティティは、プロジェクト ライフサイクルの実行フェーズにマップされます。</span><span class="sxs-lookup"><span data-stu-id="dc635-113">The Journal lines and Actuals entities are mapped to the execution phase in the project lifecycle.</span></span>

### <a name="transaction-origin"></a><span data-ttu-id="dc635-142">トランザクション発生元</span><span class="sxs-lookup"><span data-stu-id="dc635-142">Transaction origin</span></span>
