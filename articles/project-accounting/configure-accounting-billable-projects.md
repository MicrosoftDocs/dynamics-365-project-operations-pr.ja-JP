# <a name="configure-accounting-for-billable-projects"></a><span data-ttu-id="246e2-103">課金プロジェクトの会計の構成</span><span class="sxs-lookup"><span data-stu-id="246e2-103">Configure accounting for billable projects</span></span>

<span data-ttu-id="246e2-104">_**適用対象 :** リソース/非在庫ベースのシナリオに使用するプロジェクト オペレーション、見積請求に対応する小規模のデプロイ_</span><span class="sxs-lookup"><span data-stu-id="246e2-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="246e2-105">Dynamics 365 Project Operations は、時間と材料、および固定価格のトランザクションを含む、請求可能なプロジェクトのさまざまな会計オプションをサポートしています。</span><span class="sxs-lookup"><span data-stu-id="246e2-105">Dynamics 365 Project Operations supports various accounting options for billable projects that include time and material and fixed price transactions.</span></span>

- <span data-ttu-id="246e2-106">**時間と材料のトランザクション**: これらのトランザクションは、プロジェクトの時間、経費、品目、または手数料の消費に基づいて、作業の進行とともに請求されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-106">**Time and material transactions**: These transactions are invoiced as the work progresses based on the consumption of hours, expenses, items, or fees on the project.</span></span> <span data-ttu-id="246e2-107">これらのトランザクション原価は、各トランザクションの売上と照合でき、プロジェクトは作業の進行に応じて請求されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-107">These transaction costs can be matched with the revenue on each transaction and the project is invoiced as work progresses.</span></span> <span data-ttu-id="246e2-108">プロジェクトの売上は、トランザクションの発生時に見越計上することもできます。</span><span class="sxs-lookup"><span data-stu-id="246e2-108">Project revenue can be also accrued at the time when the transaction occurs.</span></span> <span data-ttu-id="246e2-109">請求時に売上が認識され、該当する場合は未収売上が取り消されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-109">During invoicing, revenue is recognized and if applicable, accrued revenue is reversed.</span></span>
- <span data-ttu-id="246e2-110">**固定価格トランザクション**: これらのトランザクションは、プロジェクト契約に基づく請求スケジュールに従って請求されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-110">**Fixed-price transactions**: These transactions are invoiced according to a billing schedule that is based on the project contract.</span></span> <span data-ttu-id="246e2-111">固定価格トランザクションの売上は、請求時に認識するか、**完了した契約** または **完了した割合** メソッドに従って定期的に計算および転記することができます。</span><span class="sxs-lookup"><span data-stu-id="246e2-111">Revenue for fixed price transactions can be recognized at invoicing or calculated and posted periodically, according to the **Completed contract** or **Completed percentage** methods.</span></span>

<span data-ttu-id="246e2-112">プロジェクトが 1 つ以上の契約品目に関連付けられている場合、そのプロジェクトは請求可能と見なされます。</span><span class="sxs-lookup"><span data-stu-id="246e2-112">A project is considered billable when it is associated with one or more contract lines.</span></span> <span data-ttu-id="246e2-113">プロジェクト契約品目は、どの請求方法およびトランザクション タイプが許可されるかを定義します。</span><span class="sxs-lookup"><span data-stu-id="246e2-113">A project contract line defines for itself which billing method and transaction types are allowed.</span></span>

<span data-ttu-id="246e2-114">一例として、Fabrikam Robotics は、機器の最適化に関する Adatum Corporation とのプロジェクト契約を受注しました。</span><span class="sxs-lookup"><span data-stu-id="246e2-114">As an example, Fabrikam Robotics has won a project contract with Adatum corporation for equipment optimization.</span></span> <span data-ttu-id="246e2-115">Adatum は、発生したプロジェクト費用をカバーするために、$10.000 USD の固定金額を支払います。</span><span class="sxs-lookup"><span data-stu-id="246e2-115">Adatum will pay a fixed amount of $10.000 USD to cover incurred project expenses.</span></span> <span data-ttu-id="246e2-116">これは固定価格の請求方法です。</span><span class="sxs-lookup"><span data-stu-id="246e2-116">This is a fixed price billing method.</span></span> <span data-ttu-id="246e2-117">プロジェクトの時間と手数料は、使用ごとに請求されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-117">Project time and fees will be billed per use.</span></span> <span data-ttu-id="246e2-118">これは時間と材料の請求方法です。</span><span class="sxs-lookup"><span data-stu-id="246e2-118">This is a time and material billing method.</span></span> <span data-ttu-id="246e2-119">すべての作業は、Adatum 機器の最適化という名前の単一のプロジェクトの下で追跡されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-119">All of the work will be tracked under a single project named, Adatum equipment optimization.</span></span>

<span data-ttu-id="246e2-120">プロジェクトチームのメンバーは、Adatum 機器の最適化プロジェクトに 8 時間の作業を提出します。</span><span class="sxs-lookup"><span data-stu-id="246e2-120">A project team member submits eight hours of work on the Adatum equipment optimization project.</span></span> <span data-ttu-id="246e2-121">プロジェクト マネージャーがこの時間を承認すると、システムは時間と材料の請求方法を使用して、実績トランザクション、請求書、および勘定を作成します。</span><span class="sxs-lookup"><span data-stu-id="246e2-121">When the project manager approves this time, the system uses the time and material billing method to create actuals transactions, an invoice, and an account.</span></span> <span data-ttu-id="246e2-122">このトランザクションは、固定価格の売上見積もりの計算には含まれません。</span><span class="sxs-lookup"><span data-stu-id="246e2-122">This transaction will not be included in the fixed price revenue estimate calculation.</span></span>

<span data-ttu-id="246e2-123">別のプロジェクト チーム メンバーが、Adatum 機器最適化プロジェクトに対して $2000.00 USD の出張費を提出します。</span><span class="sxs-lookup"><span data-stu-id="246e2-123">Another project team member submits a travel expense for $2000.00 USD against the Adatum equipment optimization project.</span></span> <span data-ttu-id="246e2-124">プロジェクト マネージャーがこの申請を承認すると、システムは固定価格の請求方法を使用して、実績トランザクションおよびこのプロジェクトの経費の勘定を作成します。</span><span class="sxs-lookup"><span data-stu-id="246e2-124">When the project manager approves this submission, the system uses a fixed price billing method to create actuals transactions and an account for this project expense.</span></span> <span data-ttu-id="246e2-125">このトランザクションに基づいて顧客に請求されることはありません。</span><span class="sxs-lookup"><span data-stu-id="246e2-125">The customer will not be invoiced based on this transaction.</span></span> <span data-ttu-id="246e2-126">代わりに、個別に構成された請求マイルストーンを使用して請求されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-126">Instead, they will be invoiced by using separately configured billing milestones.</span></span> <span data-ttu-id="246e2-127">この経費のトランザクションは、経費の見積もりとともに固定価格の売上見積もりの計算に含まれます。</span><span class="sxs-lookup"><span data-stu-id="246e2-127">This expense transaction, along with expense estimates, will be included in the fixed price revenue estimate calculation.</span></span>

<span data-ttu-id="246e2-128">プロジェクト トランザクションの会計原則は、プロジェクト コストと売上プロファイルで定義されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-128">Accounting principles for project transactions are defined in project cost and revenue profiles.</span></span> <span data-ttu-id="246e2-129">システムは、プロジェクト トランザクションごとに、構成されているプロジェクト コストと売上プロファイルのルールを使用して、適切なプロジェクト コストと売上プロファイルを決定します。</span><span class="sxs-lookup"><span data-stu-id="246e2-129">For every project transaction, the system determines the appropriate project cost and revenue profile by using the project cost and revenue profile rules that have been configured.</span></span>

## <a name="define-project-cost-and-revenue-profiles"></a><span data-ttu-id="246e2-130">プロジェクト コストと売上プロファイルの定義</span><span class="sxs-lookup"><span data-stu-id="246e2-130">Define project cost and revenue profiles</span></span> 

<span data-ttu-id="246e2-131">プロジェクト コストと売上プロファイルによって、プロジェクト トランザクションの会計ルールが決定されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-131">Project cost and revenue profiles determine the accounting rules for project transactions.</span></span> <span data-ttu-id="246e2-132">これらのプロファイルは、プロジェクト管理および会計で構成されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-132">These profiles are configured in Project management and accounting.</span></span>

       - <span data-ttu-id="246e2-144">**残高**: 時間トランザクションのコストは、勘定科目の種類、*仕掛品 - コスト値* に借方記入され、元帳転記設定の *給与配賦勘定* に貸方記入されます。</span><span class="sxs-lookup"><span data-stu-id="246e2-144">**Balance**: The cost for time transactions will be debited to the Ledger account type, *WIP - Cost value* and credited to the *Payroll allocation account* in Ledger posting setup.</span></span> <span data-ttu-id="246e2-145">経理担当者は、コストの転記機能を使用して、このコストを残高勘定から損益勘定に定期的に移動します。</span><span class="sxs-lookup"><span data-stu-id="246e2-145">The accountant will use the Post costs function to move this cost from a Balance account to a Profit and loss account on a periodic basis.</span></span>

    - <span data-ttu-id="246e2-149">**コストの転記 – 経費**:</span><span class="sxs-lookup"><span data-stu-id="246e2-149">**Post costs – expense**:</span></span>
