# <a name="labor-cost-tracking-on-projects"></a><span data-ttu-id="07473-103">プロジェクトの労務コストの追跡</span><span class="sxs-lookup"><span data-stu-id="07473-103">Labor cost tracking on projects</span></span>

<span data-ttu-id="07473-104">_**適用対象 :** リソース/非在庫ベースのシナリオに使用するプロジェクト オペレーション、見積請求に対応する小規模のデプロイ_</span><span class="sxs-lookup"><span data-stu-id="07473-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="07473-105">Dynamics 365 Project Operations では、労務費の見積もりと支出を追跡し、プロジェクト計画に必要な最小の粒度で支出します。</span><span class="sxs-lookup"><span data-stu-id="07473-105">Dynamics 365 Project Operations tracks labor estimates and spend at the lowest required granularity on the project plan.</span></span> <span data-ttu-id="07473-106">労務コストの財務上の見積もりは、計画された作業と、プロジェクト計画の各リーフ ノード タスクに割り当てられた一般的なリソースまたは名前付きリソースに基づいています。</span><span class="sxs-lookup"><span data-stu-id="07473-106">The financial estimate of labor cost is based on the planned effort and the generic or named resources that are assigned to each leaf-node task in the project plan.</span></span> <span data-ttu-id="07473-107">プロジェクトが開始され、人々がさまざまなプロジェクト タスクの時間を報告し始めると、実際の労務支出が要約され、予測の計算が開始されます。</span><span class="sxs-lookup"><span data-stu-id="07473-107">When the project begins and people start to report time on various project tasks, the actual spend on labor is summarized which starts a calculation of the projections.</span></span>

## <a name="labor-cost-tracking-view"></a><span data-ttu-id="07473-108">労務コスト追跡ビュー</span><span class="sxs-lookup"><span data-stu-id="07473-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="07473-109">**プロジェクト** ページの、**追跡** タブで、**追跡** > **原価** を選択して、**コストの追跡** ビューを開き、プロジェクト計画の各タスクに対する労務支出の進捗状況を表示および確認します。</span><span class="sxs-lookup"><span data-stu-id="07473-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="07473-110">このビューでは、タスクに費やされた労務コストの実績とタスクの計画された労務コストも比較されます。</span><span class="sxs-lookup"><span data-stu-id="07473-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="07473-111">Project Operations は、次の式を使用して労務コスト メトリックを計算します。</span><span class="sxs-lookup"><span data-stu-id="07473-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="07473-112">**計画コスト**: 各リーフ ノード タスクでのすべてのリソース割り当ての推定コスト</span><span class="sxs-lookup"><span data-stu-id="07473-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>
- <span data-ttu-id="07473-113">**実績コスト**: タスクに記録された時間のすべての実績のコストの合計</span><span class="sxs-lookup"><span data-stu-id="07473-113">**Actual Cost**: Sum of all cost actuals for time recorded on the task</span></span>
- <span data-ttu-id="07473-114">**原価消費率**：実績コスト ÷ 完了時の原価見積もり</span><span class="sxs-lookup"><span data-stu-id="07473-114">**Cost consumption percentage**: Actual cost ÷ Cost estimate at complete</span></span>
- <span data-ttu-id="07473-115">**残コスト**: 完了時のコスト - 実績コスト</span><span class="sxs-lookup"><span data-stu-id="07473-115">**Remaining Cost**: Cost estimate at complete  – Actual cost</span></span>
- <span data-ttu-id="07473-116">**完了時のコスト**: 残コスト + 実績コスト</span><span class="sxs-lookup"><span data-stu-id="07473-116">**Cost at Complete**: Remaining cost + Actual cost</span></span>
- <span data-ttu-id="07473-117">**コスト差異**: 計画コスト - 完了時のコスト予測</span><span class="sxs-lookup"><span data-stu-id="07473-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="07473-118">各タスクは、タスク上のコスト差異の推定値がタスク上に示されます。</span><span class="sxs-lookup"><span data-stu-id="07473-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="07473-119">完了時のコスト見積もりが計画コストを超える場合、タスクは予算を超えると予測されます。</span><span class="sxs-lookup"><span data-stu-id="07473-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="07473-120">完了時のコスト見積もりが計画コストを下回る場合、タスクは予算内に収まると予測されます。</span><span class="sxs-lookup"><span data-stu-id="07473-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

>[!NOTE]
> <span data-ttu-id="07473-121">Project Operations では、**追跡** タブの **プロジェクト** ページに労務コストのみが表示されます。材料と経費は消費に対して見積もって追跡できますが、これらのコストは **追跡** タブに示されているコストに含まれていません。このタブは、労力を再予測することによって労務コストを再予測するために設計されています。</span><span class="sxs-lookup"><span data-stu-id="07473-121">Project Operations only shows labor costs on the **Project** page on the **Tracking** tab. While materials and expenses can be estimated and tracked for consumption, these costs are not included in the costs shown on the **Tracking** tab. This tab is designed to work only for reprojecting labor costs by reprojecting effort.</span></span>
<span data-ttu-id="07473-122">表示されているすべての原価額は、原価率の決定に使用されたプロジェクトの原価通貨からプロジェクトの原価通貨に換算されます。</span><span class="sxs-lookup"><span data-stu-id="07473-122">All cost amounts shown are converted to the project's cost currency from the project's cost price currency used to determine the cost rate.</span></span> <span data-ttu-id="07473-123">プロジェクトの原価通貨は、プロジェクトの契約単位の通貨です。</span><span class="sxs-lookup"><span data-stu-id="07473-123">The cost currency of the project is the currency of the contracting unit on the project.</span></span> <span data-ttu-id="07473-124">**プロジェクト** ページの **見積り** タブに表示されているに時間の推定コスト値は、**追跡** タブの計画コストに含まれない可能性があります。この差異の理由は、見積もりコストは **見積り** グリッドで集計され、計画コストは **追跡** グリッドで計算される違いによるものです。</span><span class="sxs-lookup"><span data-stu-id="07473-124">The estimated cost values for time that are shown on the **Estimates** tab on the **Project** page might not add up to the planned cost on the **Tracking** tab. The reason for this discrepancy is because of the differences in how estimated cost is summarized on the **Estimates** grid and how planned cost is calculated on the **Tracking** grid.</span></span> 
>
> - <span data-ttu-id="07473-125">**見積もり** タブは、価格表のコスト率と同じ通貨を使用して、推定コストを計算します。</span><span class="sxs-lookup"><span data-stu-id="07473-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="07473-126">次に、価格表通貨での見積原価がプロジェクトの原価通貨での見積原価に換算されます。</span><span class="sxs-lookup"><span data-stu-id="07473-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="07473-127">プロジェクト通貨での推定コストは、小数点第 2 位を四捨五入して表示されます。</span><span class="sxs-lookup"><span data-stu-id="07473-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="07473-128">この換算中のどの時点でも、通貨の有効桁数は適用されません。</span><span class="sxs-lookup"><span data-stu-id="07473-128">At no point during this conversion is currency precision applied.</span></span> 
> - <span data-ttu-id="07473-129">**追跡** タブでは、計画コスト計算は、2 つの段階で適用される通貨有効桁数を含むわずかに異なる計算順序に従います。</span><span class="sxs-lookup"><span data-stu-id="07473-129">On the **Tracking** tab, the planned cost calculation follows a slightly different calculation order that involves currency precision being applied in two stages:</span></span> 
   ><ol>
   ><li><span data-ttu-id="07473-130">価格表通貨での見積原価額は、基本通貨に換算されます (換算 1)。</span><span class="sxs-lookup"><span data-stu-id="07473-130">The estimated cost amount in the price list currency is converted to the base currency (conversion 1).</span></span></li>
   ><li><span data-ttu-id="07473-131">基本通貨での見積原価額は、プロジェクト原価通貨に換算されます (換算 2)。</span><span class="sxs-lookup"><span data-stu-id="07473-131">The estimated cost amount in the base currency is converted to the project cost currency (conversion 2).</span></span> </li>
   ></ol>
   ><span data-ttu-id="07473-132">通過有効桁数は両方のステップに適用され、見積もりコスト (**見積もり** タブの **時間 - 段階的** ビュー) からわずかに逸脱した計画コスト (**追跡** タブ) を取得します。</span><span class="sxs-lookup"><span data-stu-id="07473-132">Currency precision is applied in both steps to get a planned cost (on the **Tracking** tab) that deviates slightly from the estimated cost (on the **Time - phased** view on the **Estimates** tab).</span></span> 
   
## <a name="reprojecting-costs-on-leaf-node-tasks"></a><span data-ttu-id="07473-133">リーフ ノード タスクのコストの再予測</span><span class="sxs-lookup"><span data-stu-id="07473-133">Reprojecting costs on leaf node tasks</span></span>

<span data-ttu-id="07473-134">**プロジェクト ページ** の **追跡** タブで、リーフ ノード タスクの労務コストを直接再予測することはできません。</span><span class="sxs-lookup"><span data-stu-id="07473-134">Labor costs on a leaf node task can't be directly reprojected on the **Tracking** tab on the **Project page**.</span></span> <span data-ttu-id="07473-135">ただし、**労力追跡** ビューを使って、タスクの残りの作業を再予測することができます。</span><span class="sxs-lookup"><span data-stu-id="07473-135">However, you can use the **Effort Tracking** view to reproject the remaining effort on a task.</span></span> <span data-ttu-id="07473-136">これにより、タスクの残りのコストが再計算されます。</span><span class="sxs-lookup"><span data-stu-id="07473-136">This causes a recalculation of the remaining cost on the task.</span></span> <span data-ttu-id="07473-137">次の説明はこの仕組みについてです。</span><span class="sxs-lookup"><span data-stu-id="07473-137">The following is a description of how this works.</span></span>

1. <span data-ttu-id="07473-138">プロジェクト マネージャーは、タスクの残り工数の新しい見積もりを含む **残り工数** フィールドの既定値を更新することにより、工数を再予測できます。</span><span class="sxs-lookup"><span data-stu-id="07473-138">A project manager can reproject effort on tasks by updating the default value in the **Remaining Effort** field with a new estimate of the remaining effort on the task.</span></span> <span data-ttu-id="07473-139">再予測により、タスクの完了時の工数見積もり (EAC)、進行率、およびタスクの予測された工数差異が再計算されます。</span><span class="sxs-lookup"><span data-stu-id="07473-139">Reprojecting causes a recalculation of the task's effort estimate at complete (EAC), progress percentage, and the projected effort variance on a task.</span></span> <span data-ttu-id="07473-140">サマリー タスクの EAC、完了時予測 (ETC)、および進捗率も再計算され、工数差異の新しい予測を生成します。</span><span class="sxs-lookup"><span data-stu-id="07473-140">The EAC, estimate to complete (ETC), and progress percentage on the summary tasks are also recalculated and produce a new projection of effort variance.</span></span>
2. <span data-ttu-id="07473-141">タスクの残り工数の新しい値に基づいて、システムは **コスト追跡** ビューでの残りのコストを計算します。</span><span class="sxs-lookup"><span data-stu-id="07473-141">Based on the new value for the remaining effort on the task, the system calculates the remaining cost on the **Cost Tracking** view.</span></span> <span data-ttu-id="07473-142">残りの工数に基づいて残りのコストを計算するために、システムは最初に、タスクの 1 時間の工数の平均コストを計画コストまたは計画工数として計算します。</span><span class="sxs-lookup"><span data-stu-id="07473-142">To calculate the remaining cost based on the remaining effort, the system first calculates the average cost of one hour of effort on the task as planned cost or planned effort.</span></span> <span data-ttu-id="07473-143">計画コストは、タスクのすべてのリソース割り当てのコストの合計です。</span><span class="sxs-lookup"><span data-stu-id="07473-143">The planned cost is the sum of the cost of all resource assignments on the task.</span></span> <span data-ttu-id="07473-144">1 時間あたりの平均コストは、タスクで新しく予測された残りの工数の残りのコストを計算するために使用されます。</span><span class="sxs-lookup"><span data-stu-id="07473-144">The average cost per hour is used to compute the remaining cost for the newly projected remaining effort on the task.</span></span>
3. <span data-ttu-id="07473-145">リーフ ノード タスクの完了時のコストとコスト消費の割合が再計算されます。</span><span class="sxs-lookup"><span data-stu-id="07473-145">The cost at complete and cost consumption percentage on the leaf-node task are re-calculated.</span></span>
4. <span data-ttu-id="07473-146">ルート ノードまでのサマリー タスクの完了時コスト予測が再計算されます。</span><span class="sxs-lookup"><span data-stu-id="07473-146">The cost at complete value of the summary tasks all the way to the root node are recalculated.</span></span>

## <a name="reprojecting-costs-on-summary-tasks"></a><span data-ttu-id="07473-147">サマリー タスクのコストの再予測</span><span class="sxs-lookup"><span data-stu-id="07473-147">Reprojecting costs on summary tasks</span></span>
