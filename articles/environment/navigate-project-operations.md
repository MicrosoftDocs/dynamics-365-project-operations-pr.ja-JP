 - <span data-ttu-id="654f9-106">**Common Data Service (CDS) 環境での Project Operations**: このコンポーネントは、営業案件から見積もり請求までの機能とプロセスをカバーします。</span><span class="sxs-lookup"><span data-stu-id="654f9-106">**Project Operations on Common Data Service (CDS) environment**: This component covers capabilities and processes from opportunity to proforma invoicing.</span></span>

<span data-ttu-id="654f9-108">このトピックの説明に従って Project Operations をプロビジョニングした後、Lifecycle Services (LCS) **環境の詳細** ページでは、Project Operations の両方のコンポーネントに簡単にアクセスできます。</span><span class="sxs-lookup"><span data-stu-id="654f9-108">After you provision Project Operations as described in this topic, the Lifecycle Services (LCS) **Environment details** page provides easy access to both components of Project Operations.</span></span>
