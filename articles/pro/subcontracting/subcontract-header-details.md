# <a name="header-details-for-subcontracts"></a>外注のヘッダーの詳細

_**適用対象 :** リソース/非在庫ベースのシナリオに使用するプロジェクト オペレーション、見積請求に対応する小規模のデプロイ_

この記事では、Dynamics 365 Project Operations の外注ヘッダーで提供される機能について説明します。

プロジェクト マネージャーがプロジェクトを計画および実行するとき、下請け業者を採用し、ベンダーから製品やサービスを購入する場合があります。 プロジェクト マネージャーが製品やサービスを購入する必要がある場合、Project Operations で外注を作成できます。

外注を作成するには、次の手順のようにします。

1. ナビゲーション ペインで **外注** を、**外注** ページで **新規** を選択します。
2. その他の必要情報を入力し、 **保存** を選択します。

    次のテーブルに、**外注ヘッダー** ページのフィールドに関する情報を示します。

    | フィールド | 内容 |機能的影響 |
    |---|------|---| 
    | 名称 | 外注の名前。 | すべての外注のドロップダウン リストでは、外注の名前が最初の列に表示され、外注を識別しやすくなっています。 | 
    | 内容 | 外注で購入されるサービスと製品の簡単な説明。 | いいえ​​ |
    | ベンダー | 製品およびサービスの購入元の会社の名前。 この取引先企業レコードの関係タイプは、**ベンダー** または **サプライヤー** です。 | 選択したベンダーに基づいて、次のフィールドに規定値が自動的に入力されます。<br/> **• 通貨** </br> **• 価格表** </br> **• 支払い条件**</br> **• 支払い住所**</br> **• 請求先住所**</br> **• 請求先氏名** </br>**• ベンダー取引先企業マネージャー**|
    | 外注の日付 | 外注が作成された日付。 | 外注日は、関連ベンダーに添付されている価格表またはプロジェクト パラメータから正しい購入価格表を選択するために使用されます。 |
    | ステータスの理由 | 外注の状態。 | ステータスによって、外注がビジネス プロセスのどこにあるか、および編集できるかどうかが決まります。 <br/>値は次のとおりです。<br>• **下書き**: 外注は編集できます。 状態が **下書き** の外注先のみ編集できます。<br/>• **確認済み**: ベンダーとの交渉が完了し、外注先の納入が承認されました。 <br/>• **クローズされている**: 外注先の納品が完了しました。<br/>• **キャンセル済み**: 外注はキャンセルされ、配送は予定されていません。  | 
    | 通貨 | 製品やサービスが購入された通貨。 規定値はベンダー アカウントから自動的に入力されますが、変更することができます。 | 外注の通貨は、関連ベンダーに添付されている価格表またはプロジェクト パラメータから正しい購入価格表を選択するために使用されます。 別の通貨の価格表を外注に関連付けることはできません。 ベンダー リソースがこの外注から提供する時間、費用、および資材のコストは、プロジェクトにこの通貨で記録されます。 外注レコードが保存された後は、外注の通貨を変更することはできません。|
    | 契約単位 | ベンダーと購入契約または外注契約を結んでいる会社の部門。 | いいえ​​ |
    | 支払条件 | この外注で発行されるベンダー請求書の支払条件。 規定値はベンダー アカウントから自動的に入力されますが、変更することができます。 | 外注からの支払条件は、この外注に関連するすべてのベンダー請求書にコピーされます。 外注の **下書き** のステータスが次の場合、支払条件を更新できます。 | 
    | 支払住所 | 支払いを送信すべきベンダーの住所。 規定値はベンダー アカウントから自動的に入力されますが、変更することができます。 | 外注先の支払住所は、この外注契約用に作成されたすべてのベンダー請求書に支払先住所としてコピーされます。 外注の **下書き** ステータスの場合、支払の住所を更新できます。|
    | 請求先氏名 | 請求書を送信するベンダーの会社の個人または部門の名前。 規定値はベンダー アカウントから自動的に入力されますが、変更することができます。 | 外注からの **請求先住所** の値は、この外注に関連するすべてのベンダー請求書にコピーされます。 このフィールドは、**下書き** のステータスの場合更新できます。|
    | 請求先住所 | ベンダーからの請求書で使用される住所。 規定値はベンダー アカウントから自動的に入力されますが、変更することができます。 | この住所は、この外注用に作成されたベンダー請求書の "請求元" アドレスです。 |
    | 小計 | 外注に明細がない場合は、税引前の注文の小計を入力します。 外注に品目がある場合、このフィールドは読み取り専用です。 表示される金額は、外注のすべての明細からの小計金額です。 | いいえ​​ |
    | 税額計 | 外注に明細行がない場合は、この外注の税金の総額を入力します。 外注に品目がある場合、このフィールドは読み取り専用です。 表示される金額は、外注のすべての明細行からの税金の総額です。 | いいえ​​ |
    | 合計金額 | この計算フィールドには、税金を含めた外注の合計金額が表示されます。 | いいえ​​ |
    | 確定日 | 外注が確認された日。 | いいえ​​ |
    | 依頼者 | 規定では、このフィールドは外注を作成するユーザーの名前に設定されています。 ただし、外注の作成者は、値を変更して、代理で外注を作成している人を示すことができます。 | いいえ​​ |
    | ベンダー勘定マネージャー | ベンダー アカウントの取引先責任者の名前。 規定値はベンダー アカウントから自動的に入力されますが、変更することができます。 外注先のベンダー アカウント マネージャーとして別の取引先担当者を選択できます。 | 価格交渉の結果として外注に変更が加えられたときに取引先担当者に通知するようにメール アラートを設定できます。 |
