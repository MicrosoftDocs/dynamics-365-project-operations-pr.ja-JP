# <a name="header-details-for-vendor-invoices"></a>ベンダーの請求書のヘッダーの詳細

_**適用対象:** ライト展開 - 見積もり請求の取引_

この記事では、Microsoft Dynamics 365 Project Operations のベンダー請求書ヘッダーで提供される機能について説明します。

プロジェクト マネージャーは、プロジェクトを計画および実行するときに、下請け業者を雇用し、ベンダーから製品やサービスを購入する場合があります。 プロジェクトの実行中に、ベンダーとの下請けで調達されるサービス、材料、費用のカテゴリからコストが発生します。 ベンダーは、ベンダーの請求書を作成することにより、これらのコストをプロジェクトに請求します。

次の表に、Project Operations のベンダー請求書ヘッダーのフィールドに関する情報を示します。

| Field | 説明設定 | 機能的影響 |
| --- | --- | --- |
| 件名 | ベンダー請求書の名前。 | すべてのベンダー請求書のドロップダウン リストでは、ベンダー請求書の名前が最初の列に表示され、ベンダー請求書を識別しやすくなっています。 規定では、ベンダー請求書が外注から作成される場合、ベンダー請求書の項目の **名前** フィールドは、外注先の名称と現在の日付で構成される値に設定されます。 |
| 説明設定 | ベンダーの請求書に請求されているサービスと製品の簡単な説明。 | None |
| ベンダー | 製品とサービスの請求を行っている会社の名前。 この会社は、関係タイプが **ベンダー** または **サプライヤー** のようなアカウント レコードである必要があります。 | <p>選択したベンダーに基づいて、次のフィールドに規定値が自動的に入力されます。</p><ul><li>Currency</li><li>価格表</li><li>支払条件</li><li>支払住所</li></ul> |
| 外注 | ベンダー請求書の下請けへの参照。 | <p>選択した下請けに基づいて、次のフィールドに規定値が自動的に入力されます。</p><ul><li>Currency</li><li>価格表</li><li>支払条件</li><li>支払住所</li></ul><p>ベンダー請求書ヘッダで選択された外注は、規定でベンダー請求書明細行に入力され、そこで変更することはできません。</p> |
| 請求書の日付 | ベンダー請求書が確認されたときに作成されるコスト実績の日付。 | 請求日も、関連ベンダーに添付されている価格表またはプロジェクト パラメータから正しい購入価格表を選択するために使用されます。 |
| 状態の理由 | ベンダーの請求書のステータス。 | <p>ステータスによって、ベンダー請求書がビジネス プロセスのどこにあるか、および編集できるかどうかが決まります。 こちらが利用可能な在外の一部です。</p><ul><li>**下書き** - ベンダー請求書は編集できます。</li><li>**確認済み** – ベンダー請求書が検証および確認されました。 この状態のベンダー請求書は、編集や削除をすることができません。</li><li>**処理中** – ベンダー請求書を確認しています。 この状態のベンダー請求書は、編集はできますが削除はすることができません。</li><li>**キャンセル済み** – ベンダー請求書がキャンセルされました。 この状態のベンダー請求書は、編集や削除をすることができません。</li></ul> |
| Currency | ベンダー請求書に記載されている製品およびサービスに対してベンダーが請求している通貨。 | 外注を参照するベンダー請求書では、規定で外注の通貨がベンダー請求書の通貨として入力されます。 外注を参照しないベンダー請求書では、規定値はベンダー勘定レコードから入力され、変更することができます。 ベンダー請求書が保存されると、通貨は編集できなくなります。 |
| 単位を契約中 | ベンダーからサービスや製品を受け取る責任がある会社の部門。 | None |
| 支払条件 | 発行されたベンダー請求書の支払条件。 規定値はベンダー アカウントから自動的に入力されますが、変更することができます。 | 外注からの支払条件は、この外注に関連するすべてのベンダー請求書にコピーされます。 ベンダー請求書に **下書き** のステータスがある場合、支払条件を更新できます。 |
| 支払住所 | 支払いを送信すべきベンダーの住所。 規定値はベンダー アカウントから自動的に入力されますが、変更することができます。 | 外注からの支払住所は、この外注契約用に作成されたすべてのベンダー請求書に支払先住所としてコピーされます。 ベンダー請求書に **下書き** のステータスがある場合、支払住所を更新できます。 |
| 小計 | ベンダー請求書に明細行がない場合は、税引前の請求書の小計を入力します。 ベンダー請求書に明細行がある場合、このフィールドは読み取り専用です。 この場合は、表示される金額は、ベンダー請求書上のすべての明細行からの小計金額です。 | None |
| 合計税額 | ベンダー請求書に明細行がない場合は、外注の税金の合計を入力します。 ベンダー請求書に明細行がある場合、このフィールドは読み取り専用です。 この場合は、表示される金額は、ベンダー請求書上のすべての明細行からの税金が苦の合計金額です。 | None |
| 合計金額 | この計算フィールドには、税金を含めた後のベンダー請求書の合計金額が表示されます。 | None |

> [!NOTE]
> ベンダー請求書の次のフィールドは、ベンダー請求書で **ベンダー**、**下請**、**通貨**、**契約単位**、**支払い条件** を保存した後に変更することはできません。 ベンダー請求書のこれらのフィールドに変更が必要な場合は、ベンダー請求書を削除して、新しいベンダー請求書を作成する必要があります。
