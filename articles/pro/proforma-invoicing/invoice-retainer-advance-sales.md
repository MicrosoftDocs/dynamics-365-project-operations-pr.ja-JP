# <a name="invoice-a-retainer-or-an-advance"></a><span data-ttu-id="64fb7-103">前払いまたは着手金の請求</span><span class="sxs-lookup"><span data-stu-id="64fb7-103">Invoice a retainer or an advance</span></span>

<span data-ttu-id="64fb7-104">_**適用対象 :** リソース/非在庫ベースのシナリオに使用するプロジェクト オペレーション、見積請求に対応する小規模のデプロイ_</span><span class="sxs-lookup"><span data-stu-id="64fb7-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="64fb7-105">Dynamics 365 Project Operations は着手金ベースの契約と 1 回限りの前払いをサポートします。</span><span class="sxs-lookup"><span data-stu-id="64fb7-105">Dynamics 365 Project Operations supports retainer-based contracts and one-time advances.</span></span> <span data-ttu-id="64fb7-106">プロジェクト契約では、着手金のスケジュールまたは 1 回限りの前払金を記録できます。</span><span class="sxs-lookup"><span data-stu-id="64fb7-106">On a project contract, you can record a schedule of retainers or a one-time advance.</span></span> <span data-ttu-id="64fb7-107">ただし、プロジェクト契約レベルで記録したからといって、着手金または前払金をすぐに使用できるわけではありません。</span><span class="sxs-lookup"><span data-stu-id="64fb7-107">However, recording at the project contract level doesn't immediately make a retainer or advance available for use.</span></span> <span data-ttu-id="64fb7-108">実際に顧客に請求する請求書で着手金または前払金を使用するには、最初に着手金または前払金を請求する必要があります。</span><span class="sxs-lookup"><span data-stu-id="64fb7-108">To use a retainer or advance on an invoice that actually charges the customer, the retainer or advance must be invoiced first.</span></span>

<span data-ttu-id="64fb7-109">次の手順を実行して、着手金または前払金を請求します。</span><span class="sxs-lookup"><span data-stu-id="64fb7-109">Complete the following steps to invoice a retainer or an advance.</span></span>

1. <span data-ttu-id="64fb7-110">**営業** > **請求** > **着手金または前払金** を選択します。</span><span class="sxs-lookup"><span data-stu-id="64fb7-110">Select **Sales** > **Billing** > **Retainers and Advances**.</span></span> 
2. <span data-ttu-id="64fb7-111">**前払金と着手金** ページでフィルターを使用して、請求する特定の着手金または前払金を選択し、**請求準備完了** としてマークします。</span><span class="sxs-lookup"><span data-stu-id="64fb7-111">On the **Advances and Retainers** page, use the filter to select the specific retainer or advance to invoice and mark it as **Ready to Invoice**.</span></span>

## <a name="create-a-retainer-or-advance-from-the-invoice-grid"></a><span data-ttu-id="64fb7-118">請求書グリッドから着手金または前払金を作成する</span><span class="sxs-lookup"><span data-stu-id="64fb7-118">Create a retainer or advance from the Invoice grid</span></span>

<span data-ttu-id="64fb7-119">請求書に直接、着手金または前払金を作成できます。</span><span class="sxs-lookup"><span data-stu-id="64fb7-119">You can create a retainer or an advance directly on an invoice.</span></span>

1. <span data-ttu-id="64fb7-120">請求書の下書きの **前払金と着手金** サブグリッドで、**新規** を選択して、新規の着手金または前払金を作成します。</span><span class="sxs-lookup"><span data-stu-id="64fb7-120">On a draft invoice, on the **Advances and Retainers** subgrid, select **New** to create a new retainer or advance.</span></span> 
2. <span data-ttu-id="64fb7-121">**簡易作成** ページで必要な情報を追加し、**保存** を選択します。</span><span class="sxs-lookup"><span data-stu-id="64fb7-121">On the **Quick Create** page, add the necessary information, and then select **Save**.</span></span> <span data-ttu-id="64fb7-122">着手金または前払金は、請求書に関連するプロジェクト契約で作成されます。</span><span class="sxs-lookup"><span data-stu-id="64fb7-122">The retainer or advance is created on the project contract related to the invoice.</span></span> <span data-ttu-id="64fb7-123">着手金または前払金は、自動的に **請求準備完了** としてマークされ、**請求書** ページの **前払金と着手金** サブグリッドに追加されます。</span><span class="sxs-lookup"><span data-stu-id="64fb7-123">The retainer or advance is automatically marked as **Ready to Invoice** and then added to the **Advances and Retainers** subgrid on the **Invoice** page.</span></span>
