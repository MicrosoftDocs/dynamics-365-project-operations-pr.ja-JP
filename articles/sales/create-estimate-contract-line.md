# <a name="estimate-a-projectbased-contract-line"></a>プロジェクトベースの契約品目を見積もる

_**適用対象:** リソース/非在庫ベースのシナリオ向け Project Operations_ 

Dynamics 365 Project Operations では、プロジェクトベースの契約品目は、契約品目の提供に関与する作業のコストと潜在的な収益を見積もるの際に便利な詳細があります。

プロジェクトベースの契約品目を見積もるには、プロジェクト ベースの **契約品目** の **契約品目の詳細** タブに移動します。  プロジェクトベースの契約品目で見積もりを作成するには、次の 2 つの方法があります :

   - 契約品目の詳細を手動で追加して、契約品目で直接見積もりを作成します。
   - プロジェクトとプロジェクト計画を作成し、プロジェクトとタスクをプロジェクトの契約品目に関連付けます。 これにより、契約品目に含まれるコンポーネントに基づいて、プロジェクト計画の見積もりを契約品目にインポートできるプロセスが可能となります。

## <a name="create-an-estimate-directly-on-a-projectbased-contract-line"></a>プロジェクトベースの契約品目の見積りを作成する

1. 契約品目に移動し、**契約品目の詳細** タブを選択します。このタブで作成した行は要約され、この **契約品目** の **契約金額** として表示されます。

| **トランザクション クラス** | **簡易作成** | このドロップダウンは、プロジェクトベースの契約品目の **全般** タブに含まれるトランザクション クラスのリストです。 | このフィールドは、自動的に作成されたコストに関連する契約品目の詳細を既定にします。 |

## <a name="update-prices-on-contract-line-details"></a>契約品目の詳細の価格を更新する

契約に添付されているプロジェクト価格表や契約単位の原価価格表の価格を変更した場合は、個別の契約品目詳細の価格を更新して変更内容を反映させることができます。 **契約** ページで、**再計算** を選択します。 この契約のすべての契約品目の価格がリセットされたことを通知する警告が開きます。 **はい** を選択して、販売とコスト契約品目の詳細の両方の価格を更新します。

## <a name="access-contract-line-details-for-cost"></a>コストの契約品目の詳細にアクセスする

**契約品目の詳細** タブで、グリッド内の行を選択して、サブグリッドのツールバーにアクションを表示します。 サブグリッド ツールバーの最初のアクションは **コストの詳細を開く** です。 この契約品目の詳細に関連する原価率と金額を表示するには、**コストの詳細を開く** を選択します。 

> [!NOTE]
> **コスト** の契約品目詳細のリソーシング会社、リソーシング単位、数量、日付、ロール、カテゴリの値を変更すると、**営業** の契約品目詳細に対応する値も変更されます。

## <a name="currency-on-contract-line-details-for-cost-and-sales"></a>コストと売上の契約品目の詳細で使用する通貨

**営業** の契約品目の詳細では、契約品目の詳細の開始日に有効なプロジェクト価格リストから既定の通貨を設定します。

**コスト** の契約品目の詳細は、**コスト** の契約品目の詳細の開始日に有効な契約単位の価格表から既定の通貨を設定します。

収益性の計算は、**コスト** と **営業** の契約品目の詳細の金額を環境の基準通貨に変換して、契約上の全体的な実際のマージンと推定マージンを報告します。

> [!NOTE]
> 日付に対して有効な為替レートがないため、通貨の丸め誤差やマージンの変更が発生する可能性があります。 これらの計算は、プロジェクト契約の場合は概算値としてのみ使用し、四捨五入の精度や為替レートの日付の有効性を意識する必要がある実際の法定報告書や、その他の報告書には使用しないでください。
